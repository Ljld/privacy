# Politique de confidentialité — YT Bullets

*Dernière mise à jour : mars 2026*

## Introduction

**YT Bullets** est une extension Chrome qui affiche un résumé sous forme de points clés sur les pages de vidéos YouTube. Cette page décrit quelles données sont utilisées et comment elles le sont.

## Données utilisées

L’extension utilise **uniquement** les données suivantes, strictement nécessaires à son fonctionnement :

- **Identifiant de la vidéo** : l’ID de la vidéo YouTube affichée (ex. `dQw4w9WgXcQ`), lu dans l’URL de la page.
- **Contenu de la vidéo** : le transcript (sous-titres) de la vidéo, lorsqu’il est disponible, pour générer le résumé.

Aucune autre donnée n’est collectée. En particulier, nous **ne collectons pas** :

- nom, adresse e-mail ou autre information permettant de vous identifier ;
- historique de navigation en dehors de la page YouTube en cours ;
- localisation, adresse IP ou données de géolocalisation ;
- mots de passe ou identifiants de connexion.

## Utilisation des données

- L’identifiant de la vidéo et, le cas échéant, le transcript sont envoyés à **notre serveur** uniquement pour produire la liste des points clés.
- Ces données sont traitées **en temps réel** pour afficher le résumé dans l’extension.
- Nous ne vendons pas, ne louons pas et ne partageons pas ces données avec des tiers.
- Nous n’utilisons pas ces données à des fins publicitaires, de profilage ou de crédit / solvabilité.

## Stockage

- L’extension peut utiliser le stockage local du navigateur (permission `storage`) pour conserver des **préférences** (par exemple afficher ou masquer le bloc de résumé). Aucune donnée personnelle n’y est enregistrée.
- Les données envoyées au serveur pour la génération du résumé ne sont pas conservées au-delà du traitement nécessaire à la requête, sauf si la loi nous y oblige.

## Code distant

Tout le code JavaScript de l’extension est inclus dans le package de l’extension. Aucun code n’est chargé depuis un serveur externe. Les seuls échanges avec notre serveur concernent l’envoi de l’ID de la vidéo (et éventuellement du transcript) et la réception du résumé textuel.

## Modifications

Toute modification importante de cette politique sera reflétée sur cette page, avec mise à jour de la date en haut du document.
